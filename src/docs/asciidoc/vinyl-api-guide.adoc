== 공지사항
* 모든 요청은 https 로 요청해야 합니다.
* api 추가, 수정 등 요청은 따로 문의주세요.

== 변경 이력(2021.08.28)
|===
|날짜|api|내용

|2021.08.28
|회원 가입
|회원 가입시 권한 리스트 요청 필요없음. 회원 가입하면 디폴트 권한은 ROLE_USER.

|2021.08.23
|본인 이메일 인증 요청
|개발 완료. (회원가입시 이메일 인증이 필요)

|2021.08.23
|본인 이메일 인증 결과 조회
|개발 완료. (회원가입시 이메일 인증이 필요)

|2021.08.20
|`*`
|모든 api path에 prefix로 버전 정보 추가됨.(/login 제외)
|===
